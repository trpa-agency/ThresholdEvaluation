Threshold Evaluation Timeline

2020

Threshold Cleanup

Transportation and Sustainable Communities Threshold established.

Monitoring Data Collection

Data collection starts for 2023 Threshold Evaluation, monitoring begins across multiple agencies.

2021

Threshold Evaluation Accepted

The 2019 Final Threshold Evaluation accepted by the TRPA Governing Board.

Monitoring Data Collection Continues

Monitoring data collection continues as part of the 2023 Threshold Evaluation effort.

2022

Monitoring Data Collection (Continued)

Monitoring data collection continues for the 2023 Threshold Evaluation, with further data inputs.

Evaluation Updates

Updates to the data and review process for the Threshold Evaluation begin, incorporating new findings.

2023

Threshold Evaluation Draft

2023 Threshold Evaluation draft production and Threshold Dashboard updates.

Peer Review

Evaluation peer reviews and feedback sessions, with external experts contributing insights.

2024

Threshold Standards Updated

Threshold standards for SEZ, TYC, AIS updated and finalized.

Evaluation Review

Final review of the Threshold Evaluation results and public presentation of findings.

2025

Final Report

Final 2023 Threshold Evaluation and Recommendations presented to APC and Governing Board.
